Kettlebel

Kettlebell é um equipamento utilizado no mundo antigo para exercícios musculares. Trata-se de uma bola de ferro fundido com uma alça. Os homens fortes da época usavam esses pesos para demonstrar suas forças, participando de atividades e jogos com essas peças antigas.
